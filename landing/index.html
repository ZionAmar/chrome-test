How it works

  1. Install the extension and browse as usual.
  2. Typing in any input or textarea is saved in the background (except passwords and card fields).
  3. If something was lost — click the extension icon, choose Restore, and the text is back (or copied to clipboard).

The extension does not send data to any server. Everything stays on your device.
