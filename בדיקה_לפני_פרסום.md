# בדיקה לפני פרסום — למה זה עדיין לא נותן לפרסם

החנות לא תאפשר **Submit for review** עד שכל הנקודות הבאות מסומנות. עבר על הרשימה ובדוק מה חסר.

---

## 1. כרטיסיית "רישום" / Store listing

| ✓ | מה לבדוק |
|---|-----------|
| ☐ | **תיאור** — יש טקסט ארוך (לא רק משפט אחד)? |
| ☐ | **סמל חנות** — הועלה אייקון 128×128? |
| ☐ | **צילום מסך** — לפחות תמונה אחת ב־Localized **או** Global? (1280×800 או 640×400) |
| ☐ | **קטגוריה** — נבחרה (למשל Tools)? |
| ☐ | **תוכן למבוגרים** — סומן "לא"? |

---

## 2. כרטיסיית "נוהלי פרטיות" / Privacy

| ✓ | מה לבדוק |
|---|-----------|
| ☐ | **Single purpose description** — יש טקסט (העתקת מ-PRIVACY_FORM_COPY_PASTE)? |
| ☐ | **storage justification** — יש טקסט? |
| ☐ | **activeTab justification** — יש טקסט? |
| ☐ | **Host permission justification** — יש טקסט? |
| ☐ | **Remote code** — נבחר "No"? |
| ☐ | **What user data** — סימנת לפחות משהו (למשל Website content) אם נדרש? |
| ☐ | **שלוש התיבות** "I certify that..." — שלושן מסומנות? |
| ☐ | **Privacy policy URL** — מולא עם קישור (למשל https://zionamar.github.io/chrome-test/landing/privacy.html)? |

---

## 3. חשבון (Account) — לא בתוך הפריט!

| ✓ | מה לבדוק |
|---|-----------|
| ☐ | **Contact email** — הוזן אימייל ליצירת קשר? (בדשבורד: אייקון החשבון / Settings / Account) |
| ☐ | **אימות אימייל** — קיבלת מייל מ-Google ולחצת על הקישור לאימות? |

---

## 4. איפה רואים מה חסר?

- בדף **עריכת הפריט**: לרוב יש **רשימת אזהרות** או **"Complete the following"** בראש הדף או בצד.
- אם לוחצים **Submit for review** — החנות תציג **הודעה אדומה** עם רשימת השדות החסרים.

---

## מה לעשות עכשיו

1. **גלול לראש דף עריכת הפריט** — האם יש רשימה של "נדרש" / "Complete the following"?
2. **לחץ על Submit for review** — מה **בדיוק** כתוב בהודעה האדומה?
3. **כתוב כאן** את הטקסט של ההודעה (או צלם מסך) — אז אפשר לכוון אותך לשדה הנכון.

אם תשלח את המילים המדויקות שהחנות כותבת ("X is required", "נדרש Y" וכו') — אגיד לך בדיוק איפה למלא.
